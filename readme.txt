
run the app:

To run it with fake backend, set the baseUrl in enviroment to:
  'http://localhost:4000'

then run: ng serve --open --port 4000


To run it with real mongo db backend, then set the baseUrl in enviroment to:  
  'http://localhost:60757'

  run backend project at: C:\playground\DotNetCore\webapicore2jwt

  then run front end: ng serve --open --port 4000    // or any other available port
